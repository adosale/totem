Botones

Para el proyecto tenemos los siguientes botones que se pueden usar de diferentes tamaño según sea necesario.

Aqui se tienen diferentes colores, se usa cada uno dependiedo de la situacion, creo que la mayoria solo usaremos el primario:

En codigo:

<button class="btn btn-default">Default</button>
<button class="btn btn-primary">Primario</button>
<button class="btn btn-info">Info</button>
<button class="btn btn-danger">Error</button>
<button class="btn btn-success">Exito</button>
<button class="btn btn-warning">Advertencia</button>
<button class="btn btn-inverse">Inverso</button>

Se pueden colocar de diferentes tamaños:

En codigo:

<button class="btn btn-hg btn-primary">Gigante</button>
<button class="btn btn-lg btn-primary">Grande</button>
<button class="btn btn-md btn-primary">Mediano</button>
<button class="btn btn-sm btn-primary">Pequeño</button>
<button class="btn btn-wide btn-primary">Largo</button>

Noten la clase como cambia de acorde, se puede hacer lo mismo para cada boton, solo se debe cambiar el btn-primary por el que se desee

Se pueden agrupar los botones:

Para ello se coloca un div que los englobe y que sea de clase btn-group. Ejemplo: 

<div class="btn-group">
<button class="btn btn-primary">1</button>
<button class="btn btn-primary">2</button>
<button class="btn btn-primary">3</button>
<button class="btn btn-primary">4</button>
</div>

Checkbox

Para los checkbox se usa un plugin en JS para que sea mas facil el manejo de sus eventos 

<label class="checkbox" for="checkbox1"><input type="checkbox" value="" id="checkbox1" data-toggle="checkbox"> Checkbox </label>
<label class="checkbox" for="checkbox2"><input type="checkbox" value="" id="checkbox2" checked="checked" data-toggle="checkbox"> Checkbox </label>
<label class="checkbox" for="checkbox3"><input type="checkbox" value="" id="checkbox3" data-toggle="checkbox"> Checkbox </label>
<label class="checkbox" for="checkbox4"><input type="checkbox" value="" id="checkbox4" checked="checked" data-toggle="checkbox"> Checkbox </label>

Para el control de eventos y revisar los estados:

$(':checkbox').radiocheck();
Metodos:

.radiocheck('toggle')
cambia de checked a unchecked y viceversa checked to unchecked and back

.radiocheck('check')
cambia el estado a checked

.radiocheck('uncheck')
cambia el estado a unchecked

.radiocheck('disable')
deshabilita el campo y pone estado de disabled

.radiocheck('enable')
quita el disabled del campo

.radiocheck('destroy')
destruye el cover del check

El codigo para revisar el estado:

$(':checkbox').radiocheck('check');
Eventos:

.on('change.radiocheck')
Se dispara cuando se cambia el estado del checkbox checked y unchecked.

.on('destroyed.radiocheck')
cuando el checkbox se destruye

El codigo para revisar el evento:

$(':checkbox').on('change.radiocheck', function() {
  // Inserta codigo
});

<label class="radio" for="radio4"><input type="radio" name="optionsRadios4" value="" id="radio4" data-toggle="radio"> asdfd </label>
El JS:
$(':radio').radiocheck();
Metodos:

.radiocheck('toggle')
cambia el estado de checked a unchecked y viceversa

.radiocheck('check')
cambia el estado del boton a checked

.radiocheck('uncheck')
cambia el estado del boton a unchecked

.radiocheck('disable')
pone el boton en disabled

.radiocheck('enable')
quita el disabled

.radiocheck('destroy')
destruye el boton 

$(':radio').radiocheck('check');
Eventos:

.on('change.radiocheck')
Se dispara cuando cambia el estado

.on('destroyed.radiocheck')
Se dispara cuando el boton se destruye 

$(':radio').on('change.radiocheck', function() {
// Hace algo
});

Input

Agregar .flat para quitar los bordes del input:

<input type="text" placeholder="..." class="form-control flat" />

Se pueden poner de diferentes tamaño, no deberian usar otro que no sea el default

<input type="text" class="form-control input-hg" placeholder="Gigante" />

<input type="text" class="form-control" placeholder="Default" />
<input type="text" class="form-control input-sm" placeholder="Pequeño" />

Select

Un custom select basado en: Select2 plug-in.

El componente hereda todos su estilo de los botone por lo cual se pueden usar con los mismos colores, tamaño, etc. Tambien hay soporte para selecciones multiple y de manera de bloque donde ocupa todo el espacio

Grids también están soportadas, de igual forma aquí les dejo la documentació.


En el codigo, note que se usa optgroup para los separadores: 

<select class="form-control select select-primary select-block mbl">
  <optgroup label="Perfil">
    <option value="0">Mi Perfil</option>
    <option value="1">Mis Documentos</option>
  </optgroup>
  <optgroup label="Sistema">
    <option value="2">Mensajes</option>
    <option value="3">Configuracion</option>
    <option value="4">Logout</option>
  </optgroup>
</select>

<select multiple="multiple" class="form-control multiselect multiselect-info">
  <option value="0">1</option>
  <option value="1" selected>2</option>
  <option value="2" selected>3</option>
  <option value="3" selected>4</option>

Switch

Se pueden usar switches en vez de checkbox, para este trabajo solo usaremos el circular: 

<input type="checkbox" checked data-toggle="switch" name="default-switch" id="switch-01" />
<input type="checkbox" data-toggle="switch" name="default-switch" id="switch-02" />

Al igual que los botones, pueden tener diferentes colores:

<!-- Primario / Default -->
<input type="checkbox" checked data-toggle="switch" name="default-switch-colors" data-on-color="primary" data-off-color="default" id="switch-03" />

<!-- Info / Default -->
<input type="checkbox" checked data-toggle="switch" name="info-square-switch" data-on-color="info" id="switch-04" />

<!-- Exito / Default -->
<input type="checkbox" checked data-toggle="switch" name="info-square-switch" data-on-color="success" id="switch-05" />

<!-- Advertencia / Default -->
<input type="checkbox" checked data-toggle="switch" name="info-square-switch" data-on-color="warning" id="switch-06" />

<!-- Error / Default -->
<input type="checkbox" checked data-toggle="switch" name="info-square-switch" data-on-color="danger" id="switch-07" />

Igual se pueden cambiar programando:

$('#change-color-switch').bootstrapSwitch('onColor', 'success');
$('#change-color-switch').bootstrapSwitch('offColor', 'danger');

Deshabilitado:

<input type="checkbox" checked disabled data-toggle="switch" name="custom-switch-disabled" id="switch-10" />
<input type="checkbox" checked readonly data-toggle="switch" name="custom-switch-readonly" id="switch-11" />

API para manejar el estado:

$('#disable-switch').bootstrapSwitch('disabled'); // getter
$('#disable-switch').bootstrapSwitch('toggleDisabled'); // toggle deshabilitado
$('#disable-switch').bootstrapSwitch('disabled', true);  // setter, true || false

$('#readonly-switch').bootstrapSwitch('readonly'); // getter
$('#readonly-switch').bootstrapSwitch('toggleReadonly'); // toggle readonly
$('#readonly-switch').bootstrapSwitch('readonly', true);  // setter, true || false

Se le puede cambiar el texto (debe ser corto el texto, sino queda cortado):

<input type="checkbox" checked data-toggle="switch" name="custom-switch-text" data-on-text="Hola" data-off-text="Chao" id="switch-10" />

Programandolo:

$('#label-switch').bootstrapSwitch('onText', 'Hola');
$('#label-switch').bootstrapSwitch('offText', 'Chao');

Tambien se puede hacer con iconos:

<!-- Switch cuadrado con iconos -->
<div class="bootstrap-switch-square">
  <input type="checkbox" data-on-text="<i class='fui-check'></i>" data-off-text="<i class='fui-cross'></i>" name="custom-square-switch-icons" id="switch-11" />
</div>

Etiquetas

Para manejar varias etiquetas que se necesiten. Se pueden usar en dos colores: el default y el primario.

Este componente está basado en etiquetas Bootstrap.

El codigo:

<!-- Default -->
<input name="tagsinput" class="tagsinput" data-role="tagsinput" value="Elemento, Etiqueta, Otro, Ejemplo" />

<!-- Primario-->
<div class="tagsinput-primary">
<input name="tagsinput" class="tagsinput" data-role="tagsinput" value="Ejemplo, Otro, Tercero" />
</div>

Se usa el atributo data-role="tagsinput" para inicializar el plugin 

<input name="tagsinput" class="tagsinput" data-role="tagsinput" value="Ejemplo, Otro, Tercero" />

Tambien se puede inicializar usando JS:

$(".tagsinput").tagsinput();

Cualquier cosa aquí se encuentra la documentacion oficial del plugin.

Typeahead

Rediseño del Twitter Typeahead.

Ejemplo con estados de EEUU (firma contra Obama), si escribes A te saldrán sugerencias:

Para disparar el typeahead necesitas los siguientes JS incluidos:


  • typeahead.jquery.js — vista
  • bloodhound.js — motor de sugerencias

Se puede usar typeahead.bundle.js(que contiene tanto bloodhound.js como typeahead.jquery.js)

En codigo:

<div class="form-group">
  <input class="form-control" type="text" id="typeahead-demo-01" />
</div>

El JS para el caso anterior:

var states = new Bloodhound({
  datumTokenizer: function(d) { return Bloodhound.tokenizers.whitespace(d.word); },
  queryTokenizer: Bloodhound.tokenizers.whitespace,
  limit: 4,
  local: [
    { word: "Alabama" },
    { word: "Alaska" },
    { word: "Arizona" },
    { word: "Arkansas" },
    { word: "California" },
    { word: "Colorado" }
  ]
});

states.initialize();

$('#typeahead-demo-01').typeahead(null, {
  name: 'states',
  displayKey: 'word',
  source: states.ttAdapter()
});

Recursos que le pueden servir: Ejemplos, documentacion de Typeahead, Documentacion de Bloodhound.
